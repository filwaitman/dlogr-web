About

What is Dlogr?

Dlogr is a object-oriented logging. It means that you should definitely use it to track things like your users' activities, your orders' status, etc.

Please note that Dlog is not a general purpose, traditional logging tool. Dlogr aims to provide a logging over rich objects, enriching them with events/metadata. If you want a traditional logging tool please check other amazing projects such as Papertrail, Splunk and Logentries.

How much is Dlogr?

Dlogr is absolutely free. No ads, no trial. It's an open source project also. We believe on open source power. =]

For the time being the team itself is bearing with the costs (infrastructure + development) of this project. Unfortunately, we can't maintain it like this forever so any help will be very appreciated.

How can I help you support Dlogr?

Any kind of support is extremely welcome! There are plenty ways to help us:
* Donate to the developers;
* Contribute to our project (by developing new clients, working on some new features, etc);
* Send us some feedback/suggestions.

For more details about all those kinds of support please click here.

How do I integrate my current project with Dlogr?

We receive your data through API. So you just need to send a request to us with the data you want us to track, and we'll take care of that. =)

If your project is running python, its even easier: We developed a python client so you all you need is to add on your codebase about 5 lines of code... and you're up and running with us.

Please refer to the API full documentation or to the Python client documentation for more details.
