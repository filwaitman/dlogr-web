Support us

We are free and we really want to keep this policy. However, we need some support to keep this is we want to. Below you will find ways to help us on this journey.

Donations

This is the most simple and most helpful way to support us for the time being. We need to bear with infrastructure and development costs, and if we don't get a way to share those expenses this will become a showstopper for us shortly. Any value will be appreciated.

Ah! Don't forget to let us know after a donation is made so we can put your (individual/company) name on our heroes list. =]

Development contributions

Code is power!

If you're a developer I would love to receive some help on creating new features, solving issues and developing new clients!

For now we have the following repositories on GitHub: Dlogr API, Dlogr Web client and Dlogr Python client.

Each of those repositores has its details regarding coding styles, how to set up the project and run it locally, documentation and so on. Also, we are using the GitHub issue tracker to centralize our known issues and features backlog. You can grab some issue and send us some pull request. You'll be on our heroes list as well.

Suggestions and feedback

We'd love to hear your thoughts!

If you like (or dislike) something, or have some ideas to make Dlogr even nicer, please send us an email: support@dlogr.com
